# A Collection of Interesting Github Projects

1. [Smassh](https://github.com/kraanzu/smassh)
  - basically monkeytype but for browser

2. [Webssh](https://github.com/huashengdun/webssh)
  - let you use ssh in browser
  
3. [Copyparty](https://github.com/9001/copyparty/)
  - A super fast and simple yet feature-ful portable file server
  
4. [Glances](https://github.com/nicolargo/glances)
  - system monitor in terminal and browser

5. [Matrices](https://udaign.github.io/matrices/)
  - turn any image into dot matrix

6. [Gitfetch](https://github.com/Matars/gitfetch)
  - Shows your github contributes in terminal

7. [Cargo-deb](https://github.com/kornelski/cargo-deb)
  - turn rust packages into installable deb packages
