# Interesting Website

1. [wttr.in](wttr.in)
  - Run a curl or visit this site in a browser to see the weather.
  ```bash
  # Get weather information based on geolocation information
  curl wttr.in
     
  # Get weather of a specific location
  curl wttr.in/Singapore
  curl wttr.in/Tokyo
  curl wttr.in/London
  ```

2. [Cheat.sh](cheat.sh)
  - tldr style cheatsheet for terminal command through curl
  ```bash
  # To query about usage of ls command
  curl cheat.sh/ls
  ```

3. [ident.me](ident.me)
  - query your public IP address
  ```bash
  curl ident.me
  ```

4. [BuildCores - Interactive PC Building in 3D](https://buildcores.com/)
  - lets you preview your PC builds based on chosen parts in 3D

5. [Devynpc Builder](https://devynpc.com/builder/)
  - PC build preview but for ITX build (and review by the youtuber himself)
  
6. [Bash scripting cheatsheet](https://devhints.io/bash)
  - a handy list of bash commands and examples
 
7. [Requestly](app.requestly.io)
  - let you call any public API and see the result in mutliple forms

8. [Paint.js](https://paint.js.org/)
  - Microsoft Paint but in a browser
